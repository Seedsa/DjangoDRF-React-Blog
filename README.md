# webpack-pro

基于 react + Django DRF 的个人博客

## 特性

## 技术栈

react^17.0.2 + webpack^5.47.1 + less^4.1.1 + typescript^4.3.5 + babel^7.14.8

## 使用

```
git clone https://github.com/Seedsa/DjangoDRF-React-Blog.git
npm install
npm run start
```

- 运行

```
npm run start
npm run dev
```

- 打包

```
npm run build
```

- 规范修复

```
f2elint fix
```

- 规范扫描

```
f2elint scan
```
